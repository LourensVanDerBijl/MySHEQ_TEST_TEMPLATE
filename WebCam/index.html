MySHEQ anti-cheat application for test writing.

(This is a prototype testing version, not the implementation code)

The purpose of this application software.

  1. Prevent the user from exceeding the available time limit.
  2. Prevent the user from performing any form of action on other applications or web pages while the test is in progress.
  3. Enable the webcam with JavaScript to ensure that the correct user is completing the test.
  4. Regarding space, set a random timer on the camera to capture random pictures at random times.

In this section, we can display the content of the test:

  • Downloadable media like PDF, Word etc. documentation
  • Video, Images and Audio.
  • Hyperlinks to other web pages that contain the test content
  • etc.

Test Proceedings

We need to explain to the users the proceeding of the test and the rules.
  • What will happen when the time expires
  • You will be requested to allow the webcam to record in order to complete this test.
  • (Example) the test will submit itself on the Submit button and when the time expires.
  • (Example) when you click on other applications or web pages this test will automatically be submitted
  • (Example) By clicking submit you acknowledge that mySHEQ may record your test process for control and monitoring purposes.
  • ETC.
